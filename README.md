# Wildlife Monitoring System

> Detect and classify wildlife species from camera trap images

## Overview
This project is part of a comprehensive AI/ML portfolio covering computer vision python.

## Features
- Production-ready implementation
- Well-documented code with examples
- Easy to customize and extend

## Installation

```bash
git clone https://github.com/jaychampaneri14/wildlife-monitoring-system.git
cd wildlife-monitoring-system
pip install -r requirements.txt
python main.py
```

## Requirements
See `requirements.txt` for full dependencies.
Main libraries: Computer Vision, Python, Yolo

## Usage
```bash
python main.py
```
Run with `--help` for full options.
